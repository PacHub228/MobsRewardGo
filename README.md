MobsRewardGo (1.16.5)
[RU] Уровни редкости для зомби/скелетов с уникальным лутом и уведомлениями в чате.
[EN] Rarity tiers for zombies/skeletons with custom loot and chat notifications.

🌟 Особенности / Features
Тиры: Обычный, Улучшенный (+5 HP), Легендарный (100 HP, x2.5 урон).

Уведомления: Чат-предупреждение в радиусе 30 блоков.

Лут: Кастомные таблицы выпадения.

🛠 Установка
Закиньте .jar в /plugins/.

Сервер: Spigot 1.16.5 (UTF-8).
Tech: Java 8, Spigot API.
